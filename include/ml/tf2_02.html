用tf.keras训练和构建模型

Fashion MNIST包含 70000 张灰度图像, 涵盖 10 个类别(是不是鞋子?)

import tensorflow as tf
from tensorflow import keras
fashion_mnist = keras.datasets.fashion_mnist
(train_images, train_labels), (test_images, test_labels) = fashion_mnist.load_data()
  1. 涵盖如下十个类别
# 上衣, 裤子, 套衫, 裙子, 外套, 凉鞋, 衬衫, 运动鞋, 包包, 裸靴
class_names = ['T-shirt/top', 'Trouser', 'Pullover', 'Dress', 'Coat',
               'Sandal', 'Shirt', 'Sneaker', 'Bag', 'Ankle boot']
  1. 数据大小
train_images.shape # >> (60000, 28, 28)
test_images.shape  # >> (10000, 28, 28)
  1. 将pixel的灰度(0,255)(0,255)(0,255)转换成(0,1)(0,1)(0,1)
train_images = train_images / 255.0
test_images = test_images / 255.0
  1. 看看数据长什么样子
  1. 神经网络的基本构造是层, 下面这个网络第一层将28×2828 \times 2828×28拍平成784784784, 剩下两个全连接层, 把784逐渐压缩成10个数字, 且数字范围是softmax取值的(0,1)(0,1)(0,1)之间
model = keras.Sequential([
    keras.layers.Flatten(input_shape=(28, 28)),
    keras.layers.Dense(128, activation='relu'),
    keras.layers.Dense(10, activation='softmax')
])
"""
Layer (type)                 Output Shape              Param #   
=================================================================
flatten_4 (Flatten)          (None, 784)               0         
_________________________________________________________________
dense (Dense)                (None, 128)               100480    
_________________________________________________________________
dense_1 (Dense)              (None, 10)                1290      
=================================================================
"""
  1. 构造完网络, 我们开始编译, 选择优化器, 损失函数和度量标准
model.compile(optimizer='adam',
              loss=tf.keras.losses.SparseCategoricalCrossentropy(from_logits=True),
              metrics=['accuracy'])
  1. 模型训练, 还可以指定batch_size(梯度更新所需样本), callbacks(其他回馈信息), validation_split(多少训练集被分到验证集), validation_data, shuffle(是否打乱训练集), class_weight(损失函数是否对所有标签一视同仁), sample_weight, ...
model.fit(train_images, train_labels, epochs=5)
"""

  1. 评估精度, 如果机器学习模型在新数据上的表现不如在训练数据上的表现，也就是泛化性不好，就表示出现过拟合
test_loss, test_acc = model.evaluate(test_images, test_labels)

predictions = model.predict(test_images) # shape = (10000, 10)
num_rows, num_cols = 5, 3; n = num_rows*num_cols
plt.figure(figsize=(2*2*num_cols, 2*num_rows))
for i in range(n):
  plt.subplot(num_rows, 2*num_cols, 2*i+1)
  plot_image(i, predictions, test_labels, test_images)
  plt.subplot(num_rows, 2*num_cols, 2*i+2)
  plot_value_array(i, predictions, test_labels)

def plot_image(i, predictions_array, true_label, img):
  predictions_array, true_label, img = predictions_array[i], true_label[i], img[i]
  plt.grid(False); plt.xticks([]); plt.yticks([])
  plt.imshow(img, cmap=plt.cm.binary)
  predicted_label = np.argmax(predictions_array)
  if predicted_label == true_label:
    color = 'blue'
  else:
    color = 'red'
  plt.xlabel("{} {:2.0f}% ({})".format(class_names[predicted_label],
                                100*np.max(predictions_array),
                                class_names[true_label]), color=color)

def plot_value_array(i, predictions_array, true_label):
  predictions_array, true_label = predictions_array[i], true_label[i]
  plt.grid(False); plt.xticks([]); plt.yticks([]); plt.ylim([0, 1])
  thisplot = plt.bar(range(10), predictions_array, color="#777777")
  predicted_label = np.argmax(predictions_array)
  thisplot[predicted_label].set_color('red')
  thisplot[true_label].set_color('blue')
