    # 基础语法

    常用元字符, s是whitespace的缩写

    \b 单词的开始或结束(word boundary) \d 数字(digit)
    \w Unicode字符(字母, 数字, 下划线, 汉字等) . 除换行符\n的其他字符
    \s 不可见字符(空格, 换行, 制表符等) ^ or \A 开头 $ or \Z 结尾

    限定词

    * 重复零次或更多 + 重复一次或更多 ? 重复零次或一次
    {5} 重复555次 {5,} 重复≥5\geq 5≥5次 {3,5} 重复3∼53\sim 53∼5次

    反义词/反转

    \S 匹配任意课件字符[^ \f\n\r\t\v] \B not \b \D not \d
    [^abc] 除 a、b、c 以外的任意字符 \W not \w

    字符

    [abc] a、b 或 c [a-zA-Z] a 到 z 或 A 到 Z
    [a-dm-p] a 到 d 或 m 到 p [a-z&&[def] d、e 或 f
    [a-d[m-p]] a 到 d 或 m 到 p [a-z&&[^bc]] 等于[ad-z]除了b,c
    re.A 匹配ASCII字符类，影响 \w, \W, \b, \B, \d, \D
     
    s = '123abc你好'
re.search('\D+', s).group()             # >> 123abc你好 unicode 默认
re.search('\D+', s, flags=re.A).group() # >> 123abc 只匹配ascii字符, unicode就不匹配了
    re.M 多行匹配(multi-line)
     
    s = 'aaa\r\nbbb\r\nccc'
re.findall('^[\w]+', s)             # >> ['aaa'] 单行模式, 默认
re.findall('^[\w]+', s, flags=re.M) # >> ['aaa', 'bbb', 'ccc'] 多行模式
    re.S .默认是不匹配\n换行符的, 这个就是让它匹配换行
     
    s = 'aaa\r\nbbb\r\nccc'
re.findall('^.*', s)             # >> ['aaa\r'] 默认不匹配换行
re.findall('^.*', s, flags=re.S) # >> ['aaa\r\nbbb\r\nccc']

    修饰符可以直接写在pattern里, 比如下面两个是一样的, 注意(?a)就是re.A

    re.search('(?a)\w+', s).group()
re.search('\w+', s, re.A).group()
    • 所谓贪婪，就是尽可能多的匹配某一个模式
      • *、+、{n,}这些表达式属于贪婪
    • 所谓懒惰，就是尽可能少的匹配某一个模式
      • 在贪婪的基础上加上?就是懒惰, 比如*?、+?、{n,}?
    """贪婪 vs 懒惰
"""
s = 'aabab'
re.search('a.*b', s).group()    # >> aabab
re.search('a.*?b', s).group()   # >> aab, 都是以b为结尾

    """正义 vs 反义
"""
s = '123abc你好'
re.search('\d+', s).group() # >> 123
re.search('\D+', s).group() # >> abc你好

    • 在每个大写字母前插入空格 (opens new window)
    s = 'PythonExercisesPracticeSolution'

    • 去掉字符串中带括号的内容, \( ... \)代表匹配括号 (opens new window)
    # [^)]代表匹配一切非右括号的内容
s = 'example (.com), github (.com), stackoverflow (.com)'
re.sub(r" ?\([^)]+\)", "", s)
>> 'example, github, stackoverflow'
    • 删除所有长度在1∼31\sim 31∼3之间的单词
    s = "The he quick blow fox run over the lazy dog."
shortword = re.compile(r'\W*\b\w{1,3}\b')
shortword.sub('', s)
>> ' quick blow over lazy.'
    • 把字符串中的小数都约到百分位
    s = "1.234 and 99.999; .555 12.345"
rnd = lambda match: "{:.2f}".format(float(match.group()))
re.sub(r'\d*\.\d+', rnd, s)
>> '1.23 and 100.00; 0.56 12.35'
