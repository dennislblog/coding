# 风险价值 (Value@Risk)

定义: 在市场正常波动下，某一金融资产或证券组合的最大可能损失。 更为确切的是指，在一定概率水平（置信度）下，某一金融资产或证券组合价值在未来特定时期内的最大可能损失(百度百科)。 常用的计算风险价值的方法是用历史数据去模拟, 根据在金融危机时期的回报数据去评估这些估计。

由于ppp往往设置的比较小，我们关注的因而是损失端的收益,
所以如果是long position在市场环境好的时候, VaR反而可能很高(波动大?)

# 历史模拟 (Historical Simulation)

历史模拟法的基本思想是用给定历史时间段上所观测到的市场因子的变化来表示市场因子的未来变化。 相当于做一个历史回报的柱形图去置信估计今天的损失。

VaRt+1p=−Percentile({RPF,t+1−τ}τ=1m,100p)VaR_{t+1}^p = -\text{Percentile}\Big(\{R_{PF,t+1-\tau}\}_{\tau=1}^m, 100p\Big) VaRt+1p​=−Percentile({RPF,t+1−τ​}τ=1m​,100p)

# 加权法 (Weighted Historical Simulation)

在上述历史模拟法中，每笔历史数据是等权重的。 但其实我们可以根据基金过去的业绩进行衰减加权，那么在预测未来上面会逐渐弱化历史久远的业绩表现。 每笔数据的权重，按η\etaη比率递减，且权重归一

ητ={ητ−1(1−η)/(1−ηm)}η=1m\eta_\tau = \Big\{\eta^{\tau-1}(1-\eta)/(1-\eta^m)\Big\}_{\eta=1}^m ητ​={ητ−1(1−η)/(1−ηm)}η=1m​

# 绝对/相对风险厌恶

风险厌恶测量方法

选自维基百科 (opens new window), 假设一个稳定投资可以获得404040元, 而一个风险赌注有一半几率获得100100100元, 一半几率啥都没有, 期望收益是505050元, 假设在他看来这两者没有区别的话, 我们可以认为他的效能函数满足: u(40)=(u(0)+u(100))/2=50u(40) = (u(0) + u(100))/2 = 50u(40)=(u(0)+u(100))/2=50, 也就是说此人最多愿意牺牲101010元的期望价值, 已获得稳定的收益保障.

u(c)u(c)u(c)的曲率越大(弯折程度), 越风险厌恶, 一种衡量风险厌恶的方法是"Arrow-Pratt measure of absolute risk-aversion, ARA", 定义是

A(c)=−u′′(c)u′(c)A(c) = -\frac{u^{''}(c)}{u'(c)} A(c)=−u′(c)u′′(c)​

根据dA/dcdA/dcdA/dc, 即风险厌恶同资产的关系区分, 有三种形式:

  1. CARA, 比如u(c)=1−exp⁡(−λc)⇒A(c)=λu(c) = 1 - \exp(-\lambda c) \Rightarrow A(c) = \lambdau(c)=1−exp(−λc)⇒A(c)=λ, 其风险厌恶水平一直保持在λ\lambdaλ, 不随资产增加减少而变化, 比如银行?
  2. DARA, 比如u(c)=log⁡(c)⇒A(c)=1/cu(c) = \log(c) \Rightarrow A(c) = 1/cu(c)=log(c)⇒A(c)=1/c, 随着资产ccc的增加, 更愿意冒险了
  3. IARA, 比如u(c)=c−λc2⇒A(c)=2λ/(1−2λc)u(c) = c - \lambda c^2 \Rightarrow A(c) = 2\lambda/(1-2\lambda c)u(c)=c−λc2⇒A(c)=2λ/(1−2λc), 随着资产ccc的增加, 变得更保守, 比如辛辛苦苦打拼的事业?

另一种方法通过衡量此人愿意投入的资金占当前个人总资产比率的程度(中产和富豪不太可能玩一样的赌注), "Arrow–Pratt measure of relative risk aversion, RRA", 定义是

R(c)=−u′′(c)⋅cu′(c)=A(c)⋅cR(c) = -\frac{u^{''}(c)\cdot c}{u'(c)} = A(c)\cdot c R(c)=−u′(c)u′′(c)⋅c​=A(c)⋅c

我们发现如果R(c)⊥cR(c) \perp cR(c)⊥c, 那么A(c)=1cA(c) = \frac{1}{c}A(c)=c1​, 是一个DARA效能, 同样我们把相对风险厌恶分成三类

  1. CRRA, 比如u(c)=c1−λ⇒R(c)=λu(c) = c^{1-\lambda} \Rightarrow R(c) = \lambdau(c)=c1−λ⇒R(c)=λ, 无论总资产增加多少, 都定投(每个月拿总资产10%用来投资或消费)
  2. DRRA, 比如u(c)=(c−η)1−λ−1/(1−λ)⇒R(c)=λc/(c−η)u(c) = (c-\eta)^{1-\lambda}-1/(1-\lambda) \Rightarrow R(c) = \lambda c/(c-\eta)u(c)=(c−η)1−λ−1/(1−λ)⇒R(c)=λc/(c−η), 相对厌恶水平随总资产增加而降低, 越有钱消费比例更高, 这不是作死吗?
  3. IRRA, 比如u(c)=1−exp⁡(−λc)⇒R(c)=λcu(c) = 1-\exp(-\lambda c) \Rightarrow R(c) = \lambda cu(c)=1−exp(−λc)⇒R(c)=λc, 越有钱, 投资比例下降, 比如在加薪或市场景气的时候, 还是选择每月定投100100100元

# 货币供应量

2021年一月最新数据(人民币单位)

M0M_0M0​ 流通中现金 9.199.199.19 Trillian
M1M_1M1​ M0M_0M0​ + 企业活期存款 59.3559.3559.35 Trillian
M2M_2M2​ M1M_1M1​+准货币（定期存款+居民储蓄存款+其它存款） 223.6223.6223.6 Trillian
  • M0M_0M0​是流通于银行体系以外的现钞, 就是居民和企业手中的现钞, 流动性最强，具有最强的购买力
  • M1M_1M1​由银行体系以外的现钞M0M_0M0​和银行的活期存款构成, 代表了一国经济中的现实购买力
  • M2M_2M2​由流通于银行体系之外的现钞加上活期存款M1M_1M1​，再加上定期存款、储蓄存款等构成. 经过一定的时间和手续后，也能够转变为购买力，因此，它们又叫做"准货币". 比如像刺激消费, 可以通过降低准货币向货币的转变成本(取消手续费,支付宝转帐等),
  • PMI=订单×30%+生产×25%+雇员×20%+配送×15%+存货×10%
  • CPI=物价消费指数, 剔除食品及能源的核心CPI是衡量一般消费品和劳务价格的重要指标。 PPI-CPI对于GDP的预测有一定意义
